# JOURNAL
Trading Journal. Trading Notes. Trading All.

# DataBase
Our DataBase working with phpMyAdmin(MySQL) on XAMPP. To start DataBase part, please open XAMPP and turn on Apache and MySQL.
You can chek this here: [localhost/phpmyadmin](http://localhost/phpmyadmin).

# BACK-END
To start back-end part, please write in terminal VS CODE project folder 'backend' this text: `uvicorn main:app --reload`
Back-end will work. You can chek this here: http://127.0.0.1:8000/

# FRONT-END
To start front-end part, please write in terminal VS CODE project folder 'frontend' this text: `ng serve`
Front-end will work. You can chek this here: http://127.0.0.1:4200
